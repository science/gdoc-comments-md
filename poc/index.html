Google Docs Comments Extractor

OAuth 2.0 Proof of Concept

Setup Instructions

  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable Google Docs API and Google Drive API
  4. Create OAuth 2.0 credentials (Web application type)
  5. Add http://localhost:8080 to Authorized JavaScript origins
  6. Copy the Client ID and paste it below
  7. Configure OAuth consent screen (add yourself as test user)
